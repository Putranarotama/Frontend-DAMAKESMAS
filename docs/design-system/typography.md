# DAMAKESMAS Typography Guide

## Font Families
- **Primary**: `Inter` (body, tables, badges, nav links)
- **Display**: `Plus Jakarta Sans` (headings, hero text, section titles)

## Type Scale
| Token | Font | Size | Weight | Line-Height | Usage |
|-------|------|------|--------|-------------|-------|
| display-xl | Plus Jakarta Sans | 48px | 800 | 1.1 | Hero headline |
| display-lg | Plus Jakarta Sans | 36px | 700 | 1.2 | Section H1 |
| heading-lg | Plus Jakarta Sans | 22px | 700 | 1.3 | Section titles |
| heading-md | Plus Jakarta Sans | 18px | 600 | 1.4 | Card titles |
| body-lg | Inter | 16px | 400 | 1.6 | Main body |
| body-md | Inter | 14px | 400 | 1.5 | Table data, secondary |
| body-sm | Inter | 12px | 400 | 1.4 | Captions, labels |
| label | Inter | 11px | 600 | 1.2 | Nav menu (uppercase) |
| stat-number | Plus Jakarta Sans | 24px | 700 | 1 | Service count numbers |
| btn-text | Inter | 14px | 600 | 1 | Button labels |

## Letter Spacing
- Nav links: `0.04em` (uppercase)
- Section labels: `0.08em`
- Body: `0`
